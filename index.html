Що таке система контролю версій?

Система контролю версій (VCS) дозволяє відслідковувати зміни у коді або файлах проекту. Git — одна з найпопулярніших розподілених VCS, яка використовується для відслідковування змін в коді, ведення історії змін і роботи з гілками.

Базові команди Git

  1. git init - Ініціалізація нового репозиторію Git в поточній папці.
  2. git clone [url] - Клонування існуючого репозиторію з віддаленого сервера.
  3. git status - Перевірка стану файлів у робочій директорії.
  4. git add [файл] - Додавання файлу до індексації (для наступного коміту).
  5. git commit -m "[повідомлення]" - Збереження змін у репозиторії з коментарем.
  6. git push - Відправка локальних змін на віддалений репозиторій.
  7. git pull - Оновлення локальної версії з віддаленого репозиторію.
  8. git branch - Перегляд гілок у репозиторії.
  9. git checkout [назва гілки] - Перемикання між гілками.
  10. git merge [назва гілки] - Злиття іншої гілки в поточну.

Коротка інформація про ОС GNU/Linux

GNU/Linux — це вільна та відкрита операційна система, яка базується на ядрі Linux і програмному забезпеченні GNU. Вона використовується на серверах, робочих станціях, вбудованих системах та багатьох інших пристроях. Відомі своєю стабільністю, безпекою та великою спільнотою розробників.

Популярні дистрибутиви GNU/Linux

  • Ubuntu — дистрибутив, орієнтований на користувачів-початківців та офісне використання.
  • Debian — стабільний дистрибутив, який часто використовується на серверах.
  • Fedora — інноваційний дистрибутив з новими технологіями.
  • Arch Linux — дистрибутив для досвідчених користувачів, який потребує ручної конфігурації.
  • CentOS — дистрибутив, популярний на корпоративних серверах завдяки стабільності.

Приклади команд роботи в Linux CLI

  1. pwd — Показує поточний каталог.
  2. echo — Виводить текст у термінал.
  3. ls — Виводить список файлів у поточному каталозі.
  4. cd — Змінює поточний каталог.
  5. touch — Створює порожній файл.
  6. mkdir — Створює новий каталог.
  7. cp — Копіює файли або каталоги.
  8. mv — Переміщує або перейменовує файли або каталоги.
  9. rm — Видаляє файли або каталоги.
  10. history — Виводить історію команд, виконаних у терміналі.
