Вивчення Linux

Коротка інформація про ОС GNU/Linux

GNU/Linux — вільна операційна система, відома своєю стабільністю, безпекою та великою спільнотою розробників.

Популярні дистрибутиви Linux

  • Ubuntu — для користувачів-початківців.
  • Debian — стабільний і надійний для серверів.
  • Fedora — інноваційний дистрибутив.
  • Arch Linux — для досвідчених користувачів.
  • CentOS — популярний у корпоративних середовищах.

Приклади команд роботи в Linux CLI

  1. pwd — Показує поточний каталог.
  2. echo — Виводить текст у термінал.
  3. ls — Виводить список файлів.
  4. cd — Змінює каталог.
  5. touch — Створює файл.
  6. mkdir — Створює каталог.
  7. cp — Копіює файли.
  8. mv — Переміщує файли.
  9. rm — Видаляє файли.
  10. history — Виводить історію команд.

BASH-скрипт авторизації


#!/bin/bash

# Заздалегідь визначені правильні ім'я користувача та пароль
correct_username="admin"
correct_password="password123"

# Функція для введення пароля з прихованими символами
function read_password {
    prompt="$1"
    stty -echo
    printf "%s" "$prompt"
    read password
    stty echo
    printf "\n"
}

# Запит на введення імені користувача
echo "Вітаємо! Будь ласка, увійдіть в систему."
read -p "Ім'я користувача: " username

# Запит на введення пароля (символи не відображаються)
read_password "Пароль:"

# Перевірка введених даних
if [[ "$username" == "$correct_username" && "$password" == "$correct_password" ]]; then
    echo "Авторизація пройшла успішно. Вітаємо, $username!"
else
    echo "Невірне ім'я користувача або пароль. Спробуйте ще раз."
fi
